## Тема баклаврской работы: "Разработка web-приложения для оценки изменений в корпусе новостных сообщений."
Разработано веб-приложение для анализа тематических изменений в новостных корпусах. Программа извлекает ключевые слова и фразы из текстов с использованием методов TF, TF-IDF и RAKE, строит тренд-карты и матрицы косинусного сходства, позволяя отслеживать динамику и значимость обсуждаемых тем. Приложение реализовано на языке R с использованием фреймворка Shiny и протестировано на региональных новостях за 2020–2024 годы.

Код программы находится в файле Result_with_UI_5tabs.R.

Описание работы есть в файле "Автореферат_диплома.pdf", описание приложения в "Описание_приложения.pdf", а результаты его работы в "Результаты работы программы.pdf".
